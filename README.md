# VEX Robotics Pushback – Control Code

This repository contains the control system code developed for my VEX Robotics Pushback season robot.

## Overview

I worked as the primary programmer for my team, developing autonomous routines and driver control systems using sensor feedback (inertial, rotation, and color sensors). The code focuses on efficient path planning and reliable autonomous performance during 15-second and 60-second autonomous periods.

## Key Contributions

- Developed multi-sensor autonomous routines with path planning
- Implemented driver control improvements for precision movement
- Worked with team members to iterate robot behavior and tuning
- Integrated sensor feedback for navigation and object interaction

## Notes

This repository is primarily used as a code archive from the competition season and may not be actively maintained.
